# Simple Banking System (Java)

## Project Description
This is a console-based Java application developed as part of the Java Developer Internship (Week 2 Task). The project implements a simple banking system where users can perform basic banking operations using core Java concepts.

## Features
* Create a bank account
* Deposit money
* Withdraw money
* Check account balance
* Data is stored permanently using file handling

## Technologies Used
* Java
* Object-Oriented Programming (Encapsulation)
* File Handling
* Exception Handling

## How to Run
1. Open terminal or command prompt.
2. Compile the Java file:
   javac BankingSystem.java
3. Run the program:
   java BankingSystem

## Data Storage
Account details are saved in a text file (accounts.txt) so that records are not lost after the program stops.
